### 🚀 Featured Projects

#### 🛒 [YFixIt](https://github.com/Goddii/yfixit) — Second-Hand Electronics Marketplace
A full-stack marketplace for broken, second-hand, and repairable electronics — built for Kenyan buyers and sellers.
- **Stack:** React + Vite (frontend) · Flask + SQLAlchemy + JWT (backend) · SQLite · Safaricom M-Pesa Daraja sandbox
- Buyer/seller roles with JWT auth, listing CRUD, search/filter/sort, and a full **M-Pesa STK Push checkout flow** with payment callbacks and order status polling
- Real-world payment lifecycle: OAuth token → STK Push → callback → order completion → listing marked sold

#### 🚚 [NextFleet](https://github.com/Goddii/fleet-management-hub) — Fleet Management System
A role-based fleet management web app for tracking trucks, drivers, jobs, and maintenance.
- **Stack:** Flask · MySQL · Jinja2 · Chart.js
- Manager/driver RBAC, mileage & fuel tracking with charts, image upload pipeline with location tagging, full job management CRUD
- Dockerized with `docker-compose` for easy local spin-up

#### 🏠 [Yetu Housing Agency](https://github.com/Goddii/yetu-housing-agency) — Real Estate Listings Platform
A team-built real estate platform for browsing property listings and connecting with agents.
- **Stack:** React 19 · React Router v7 · JSON Server · Vite
- Built collaboratively (4-person bootcamp team) — I contributed to data layer setup, filtering logic, and routing
- Property browsing with location/price/type filters, agent profiles, and a working contact form

#### 📚 [Chapters & Chai](https://github.com/Goddii/chapters-and-chai) — CLI Book Club Tracker
A command-line library tracker for logging books, tracking reading progress, and storing reviews — built from scratch in Python.
- **Stack:** Python · `argparse` · `rich` · JSON persistence
- Clean OOP design with one-to-many model relationships (`Member → Book → Review`), natural-language date parsing for due dates, and a full CLI command set

#### 🎬 [GanbareMovies](https://github.com/Goddii/movie-viewer) — Movie Viewer Web App
A frontend project for browsing and viewing movie info with a clean, responsive interface.
- **Stack:** HTML5 · Bootstrap · Vanilla JavaScript
- Dynamic content rendering and responsive UI — an early frontend fundamentals project
